Инструкции по установке
Драйвер Microsoft JDBC 4.2 для SQL Server


ВВЕДЕНИЕ
------------
Инструкции для распаковки sqljdbc_<version>_enu.exe и 
sqljdbc_<version>_enu.tar.gz.


WINDOWS
-------
1. Прочтите файл license.txt. Загрузка и использование драйвера 4.2 Microsoft JDBC для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<version>_enu.exe во временный каталог.

3. Запустите файл sqljdbc_<version>_enu.exe

4. Введите каталог установки по запросу. Рекомендуется распаковать
   этот ZIP-файл в папку % ProgramFiles % в каталоге по умолчанию:

	Драйвер Microsoft JDBC 4.2 для SQL Server


UNIX
----
1. Прочтите файл license.txt. Загрузка и использование драйвера 4.2 Microsoft JDBC для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<version>_enu.tar.gz во временный каталог.

3. Чтобы распаковать сжатый файл tar, перейдите в каталог, в котором требуется
   распаковать драйвер и введите:

	gzip -d sqljdbc_<version>_enu.tar.gz

4. Чтобы распаковать TAR-файл, переместите его в каталог, в который нужно установить драйвер 
   и введите:

	tar -xf sqljdbc_<version>_enu.tar
